This folder is specified as a static folder in the web server.
It's supposed to be used with the resource server parameter of the client.

Use this let the client access the files in this folder:

    resourcesvr:"http://<address>/user/resources/"

For more information see the Wiki, at:
http://aura-project.org/wiki/index.php/Parameters#resourcesvr
